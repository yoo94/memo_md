# js 메서드 종류

# 📚 자바스크립트 주요 메서드 종류 정리

## 🔁 배열(Array) 관련 메서드

| 메서드       | 설명                             |
|--------------|----------------------------------|
| `for...of`   |배열 요소 순회 Array, Map, Set, String 등|
| `for...in`   |객체의 키 순회       Object        | 
| `forEach()`  | 배열 순회 (반환값 없음)          |
| `map()`      | 배열 변환 (새 배열 반환)         |
| `filter()`   | 조건에 맞는 요소 추출            |
| `find()`     | 조건에 맞는 첫 요소 반환         |
| `reduce()`   | 누적 계산                        |
| `some()`     | 하나라도 조건 만족하면 true      |
| `every()`    | 모두 조건 만족해야 true          |
| `includes()` | 특정 값 포함 여부 확인           |
| `indexOf()`  | 특정 값의 인덱스 반환            |
| `splice()`   | 배열 수정 (삭제/추가/교체)       |
| `slice()`    | 배열 복사 (원본 유지)            |
| `sort()`     | 배열 정렬                        |
| `reverse()`  | 배열 순서 뒤집기                 |
| `concat()`   | 배열 합치기                      |
| `join()`     | 배열을 문자열로 변환             |
| `flat()`     | 다차원 배열 평탄화               |

---

## 🧱 객체(Object) 관련 메서드

| 메서드               | 설명                           |
|----------------------|--------------------------------|
| `Object.keys()`      | 키 배열 반환                   |
| `Object.values()`    | 값 배열 반환                   |
| `Object.entries()`   | [키, 값] 배열 반환             |
| `Object.assign()`    | 객체 복사 및 병합              |
| `hasOwnProperty()`   | 특정 키 존재 여부 확인         |

---

## 🔤 문자열(String) 관련 메서드

| 메서드         | 설명                             |
|----------------|----------------------------------|
| `charAt()`     | 특정 인덱스 문자 반환            |
| `includes()`   | 특정 문자열 포함 여부 확인       |
| `indexOf()`    | 특정 문자열 위치 반환            |
| `slice()`      | 문자열 일부 추출                 |
| `substring()`  | 문자열 일부 추출                 |
| `split()`      | 문자열을 배열로 분리             |
| `trim()`       | 앞뒤 공백 제거                   |
| `toUpperCase()`| 대문자로 변환                    |
| `toLowerCase()`| 소문자로 변환                    |
| `replace()`    | 문자열 치환                      |

| 메서드         | 설명                             |

## ⏱️ 시간(Date) 관련 메서드

| 메서드         | 설명                             |
|----------------|----------------------------------|
| `Date.now()`   | 현재 시간 (타임스탬프)           |
| `getFullYear()`| 연도 반환                        |
| `getMonth()`   | 월 반환 (0~11)                   |
| `getDate()`    | 일 반환                          |
| `getDay()`     | 요일 반환 (0: 일요일)            |

| `getSeconds()` | 초 반환                          |

# 🧮 자바스크립트 Math 객체 메서드 정리

## 📌 주요 메서드

| 메서드              | 설명                     | 예시 코드               | 반환값        |
|------------------|------------------------|---------------------|------------|
| `Math.abs(x)`    | 절댓값 반환                 | `Math.abs(-5)`      | `5`        |
| `Math.ceil(x)`   | 올림값 반환                 | `Math.ceil(4.2)`    | `5`        |
| `Math.floor(x)`  | 내림값 반환                 | `Math.floor(4.9)`   | `4`        |
| `Math.round(x)`  | 반올림값 반환                | `Math.round(4.5)`   | `5`        |
| `Math.max(...x)` | 최댓값 반환                 | `Math.max(1, 3, 2)` | `3`        |
| `Math.min(...x)` | 최솟값 반환                 | `Math.min(1, 3, 2)` | `1`        |
| `Math.random()`  | 0 이상 1 미만의 난수 반환       | `Math.random()`     | `0.123...` |
| `Math.sign(x)`   | 부호 반환 (`-1`, `0`, `1`) | `Math.sign(-10)`    | `-1`       |
